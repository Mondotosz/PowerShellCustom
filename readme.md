### PowerShell module

Run as administrator

```ps1
Install-Module -Name "PSWriteColor"
```

### Set Shell config

Copy Profile.ps1 to $HOME\Document\WindowsPowerShell\

```ps1
cp ./Profile.ps1 $HOME\Document\WindowsPowerShell\Profile.ps1
```
